# py-http-packages
Python HTTP packages comparison

## Instructions

Install [pyenv](https://github.com/pyenv/pyenv) and [PDM](https://pdm.fming.dev/latest/).

`pdm install`

`pdm run start-server # On a terminal 1`

`pdm run test-it # On a terminal 2`
